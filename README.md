# REST_API_CUSTOMERS_PROJECT

This project is a **backend API for managing customers**, built with **Bun / Node.js**, and using **MySQL** for data storage. The project is ready to run with **Docker**.

---

## Features

- Create a customer 
- Retrieve a customer or all customers 
- Search customers 
- Update a customer 
- Delete a customer and related data 
- Field validation (regex for names, identity number, age)
- Transactions for addresses and phone numbers

---

## Requirements

- Docker & Docker Compose
- Bun or Node.js
- MySQL 8
