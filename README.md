# REST_API_CUSTOMERS_PROJECT

A **backend API** for managing customer data, built with **Bun / Node.js**, **MySQL**, and containerized with **Docker**. 
---

## Tech Stack

- **Runtime / Framework**: Bun (Node.js compatible)
- **Database**: MySQL 8
- **Containerization**: Docker & Docker Compose
- **Language**: TypeScript
- **Tools**: Postman

---

## Features

- **Create a customer**
- **Retrieve a customer or all customers**
- **Search customers**
- **Update a customer**
- **Delete a customer and related addresses & phone numbers**
  
---

## Requirements

- Docker & Docker Compose
- Bun (or Node.js)
- MySQL 8
